Задача динамического переназначения воздушных судов представляет собой задачу дискретной оптимизации, возникающую в контексте оперативного управления авиаперевозками при наличии сбоев и непредвиденных событий. 

В классической форме задача заключается в том, чтобы для заданного множества рейсов и воздушных судов перестроить имеющееся назначение (отображение рейсов на ВС), так, чтобы оно удовлетворяло ряду жестких ограничений и было как можно больше похоже на исходное. В данной работе рассматривается динамическое перестроение существующего расписания в ответ на триггеры трех типов: смена конфигурации, временный запрет и сдвиг расписания.
